godtools-api
============

A java API to store, maintain and serve God Tools content to the God Tools android and iOS apps.

The API is now serving content for mobile versions of the booklets:  "Knowing God Personally", "The Four Spiritual Laws" and "Satisfied?" in 25+ languages.

The Android app using this API was released 1/27/15 w/ iOS coming soon.  Check the Google Play Store description for more highlights and news about the project.
